## Devcamper API

> Backend API for Devcamper application,
> which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env"
and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in Dev mode
npm run dev

# Run in prod mode
npm start

```
